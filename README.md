# Ashok Tech Website

A simple static website for Ashok Tech, an IT solutions company.

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styling
- `script.js`: JavaScript for interactivity
- `Website.ts`: (Currently empty, can be used for TypeScript code if needed)

## How to View

Open `index.html` in a web browser to view the website.

## Features

- Responsive design
- Navigation with smooth scrolling
- Services section
- Contact information

## Development

To modify the website, edit the HTML, CSS, and JS files directly.
